## 📝 서비스 소개
- 우리는 현재 대한민국의 유행 중 하나인"독서"를 활성화 하고 우울감에 빠진 현대인들이 정서를 "독서"를 통해 치유하고 더불어 나아가, 자극적인 콘텐츠에 따른 도파민 중독에 빠진 사람들을 독서로 힐링할 수 있게 하고자 해당서비스" 별책부록" 을기획하게 되었습니다.
